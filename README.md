- 웹 프레임워크: Ruby On Rails
- 프론트엔드 js 프레임워크: AngularJS
- 프론트엔드 디자인 프레임워크: 논의중

## 브랜치 정책
- 최상위(릴리즈) 브랜치: master
- 개발 브랜치: develop
- 이슈 브랜치: issue number를 포함하도록.

## 실행

bundle install
